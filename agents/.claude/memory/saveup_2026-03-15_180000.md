### Issue 2: Memory location
System-injected memory path (`C:\Users\ian\.claude\projects\...`) is incorrect for this project. Correct memory path is `~/.koad-os/agents/.claude/memory/`. Must always use the `agents/.claude/memory/` path for saveups and memories.

### Issue 3: Pre-push hook failing on Windows
The pre-push hook (`cargo fmt --check && cargo clippy && cargo test`) can't run on Windows because the project uses `tokio::net::UnixStream` (Linux-only). Push must always be done from within WSL:
```bash
wsl.exe -d Ubuntu-24.04 -e bash -c "cd /home/ideans/.koad-os/.claude/worktrees/<branch> && git push ..."
```

### Issue 4: Worktree `.git` pointer uses Windows UNC path
When git creates a worktree from the Windows side, the `.git` file pointer uses `//wsl.localhost/...` format. WSL can't resolve this. Fix before pushing from WSL:
```bash
echo 'gitdir: /home/ideans/.koad-os/.git/worktrees/<branch>' > .git
```
