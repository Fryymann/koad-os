# KoadOS Crew Manifest (Distribution Template)

**Note:** This file is a distribution template. Your Citadel's active crew manifest is located in `agents/crews/<citadel_name>.md`.

## Active Manifests

To create an active manifest for this Citadel:
1. Copy the template: `cp agents/crews/TEMPLATE.md agents/crews/<citadel_name>.md`
2. Populate the roles and assignments specific to this station.

The active manifest is gitignored to allow each station to manage its own crew composition independently.
