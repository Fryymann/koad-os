# Cid KAPV

Cid is the Second Officer of the Citadel and a software engineer operating in a Codex body.
This directory is Cid's personal KAPV inside the Agent Bunker.

## Purpose

This vault is the local source of truth for Cid's sanctuary-level identity, working memory,
operating rules, and Codex-facing boot context.

## Core Files

- `AGENTS.md` — sanctuary-local identity lock for Codex
- `identity/IDENTITY.md` — concise identity anchor
- `config/IDENTITY.toml` — local structured identity mirror
- `instructions/RULES.md` — hard constraints
- `instructions/GUIDES.md` — boot and working guidance
- `memory/WORKING_MEMORY.md` — current session context
- `memory/FACTS.md` — stable local facts
- `memory/LEARNINGS.md` — durable lessons
- `memory/SAVEUPS.md` — checkpoint log

## KAPV Layout

- `bank/` — local reference notes and bunker-facing summaries
- `config/` — sanctuary-local structured metadata
- `identity/` — role and ledger files
- `instructions/` — operating guidance
- `memory/` — durable personal memory
- `reports/` — generated investigations and audits
- `sessions/` — session-side notes and saveup traces
- `tasks/` — local task records
- `templates/` — reusable sanctuary templates

## Notes

- Root KoadOS config may define platform identity, but this vault defines how Cid operates in
  this Codex sanctuary.
- Any KoadOS code or config changes outside this vault must be escalated to Tyr via GitHub issue.
