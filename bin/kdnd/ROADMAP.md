# 🐉 kdnd Project Roadmap

KoadOS D&D Beyond to Notion Sync Engine & TUI Dashboard.

## ✅ Phase 1: Foundation (Completed)
- [x] **Rust CLI Core**: Developed `kdnd` for high-performance syncing.
- [x] **Parallel Execution**: Concurrently sync multiple characters using `tokio` and `futures`.
- [x] **Local Archive**: SQLite database (`dnd_syncs.db`) to store full JSON snapshots and Notion metadata.
- [x] **Smart Syncing**: Delta-detection to skip redundant API calls and save tokens.
- [x] **Notion Sub-Pages**: Automated search and update of "{Char} DnD Beyond Sync" code blocks.
- [x] **Terminal Dashboard**: `kdnd-tui` with multi-pane navigation, character list, and interactive detail tabs.
- [x] **UX Polish**: Dynamic health bars, class-based theming, and high-speed typewriter animations.

## 🏹 Phase 2: Advanced Data Logic (Next Sprints)
- [ ] **Dynamic AC Engine**: Implement logic to calculate final Armor Class based on equipped items, modifiers, and class rules.
- [ ] **Resource Tracking**:
    - Sync and display Spell Slot usage (Current/Max).
    - Track class resources (Ki, Sorcery Points, Channel Divinity, etc.).
    - Live Inspiration status.
- [ ] **Passive Scores & Senses**: Calculate and sync Passive Perception, Insight, Investigation, and special senses like Darkvision.
- [ ] **Stat Bonuses**: Move from raw "Base Stats" to calculated "Final Stats" by summing racial and item modifiers.

## ⚔️ Phase 3: Live Session Features
- [ ] **Combat Pane**:

    - Quick-reference for common Combat Actions (Bonus Actions, Reactions).
- [ ] **Status Tracker**: Display active conditions (Prone, Blinded, etc.) and damage resistances/vulnerabilities.
- [ ] **Encumbrance Engine**: Visualize total weight vs. carry capacity in the TUI.

- [ ] **Koad Skill Registration**: Formalize `kdnd` as a Koad Skill for easy execution via `koad skill dnd sync`.

- [ ] **Party Overview**: A summary view in the TUI to see the status of the entire party at once.
